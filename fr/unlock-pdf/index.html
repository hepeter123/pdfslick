How PDF Password Protection Works

PDF files support two types of passwords: a user password (also called an open password) that prevents opening the document, and an owner password (also called a permissions password) that restricts actions like printing, copying text, or editing. When you "unlock" a PDF, you're creating a new copy of the document with the encryption removed, so the content can be accessed without any password going forward. Note that you must know the correct password to unlock the file — this tool does not crack or bypass unknown passwords.
